Datenschutz (Projekt/Demo)

Diese Website ist ein Demo-Shop. Es werden keine Daten an einen Server gesendet. Optional speichern wir Eingaben lokal im Browser über localStorage.

LocalStorage

Für Warenkorb, Rechnungsadresse und Bestellinfos nutzt die Seite localStorage. Diese Daten bleiben auf deinem Gerät und können jederzeit gelöscht werden.

Wie lösche ich die Daten?

Browser → Einstellungen → Website-Daten löschen oder im Shop über “Warenkorb leeren”.

Keine echten Zahlungen

Es gibt kein Payment-Gateway. Alle Preise sind fiktiv.
