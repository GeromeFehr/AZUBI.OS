Datenschutzerklärung (Demo / Schulprojekt)

Diese Website ist ein Schulprojekt (Demo-Shop) und verarbeitet keine echten Bestellungen. Es wird kein Backend betrieben und keine Daten an einen Server gesendet.

1. Verantwortlicher

Verantwortlich für diese Demo-Seite:
Jotham, Noah, Gérôme (Projektteam AZUBI.OS)
Kontakt: Bitte im Impressum ergänzen

2. Welche Daten werden verarbeitet?

Der Demo-Checkout kann eine Rechnungsadresse abfragen. Diese Angaben werden – wenn aktiviert – nur lokal im Browser über localStorage gespeichert, damit der Shop-Flow demonstrierbar ist. Eine Übertragung an Server findet nicht statt.

3. localStorage (Browser-Speicher)

localStorage ist ein Speicherbereich im Browser. Er dient hier ausschließlich dazu, Warenkorb/Adresse/Bestellung im Demo-Flow zu halten. Du kannst die Daten jederzeit löschen: Browser-Einstellungen → Website-Daten → Seite auswählen → Daten löschen.

4. Cookies / Tracking

Diese Demo-Seite nutzt keine Tracking-Cookies, kein Analytics und keine Werbe-Tracker. (Wenn du später Analytics nutzt, musst du das hier ergänzen.)

5. Drittanbieter / GitHub Pages

Die Seite wird über GitHub Pages ausgeliefert. Dabei kann GitHub technisch notwendige Zugriffsdaten (z.B. IP-Adresse, User-Agent) in Server-Logs verarbeiten. Details findest du in den GitHub-Datenschutzinformationen.

Hinweis: Diese Datenschutzerklärung ist eine Projekt-/Demo-Vorlage und keine Rechtsberatung.
